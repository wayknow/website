The Mac App Store is full of color pickers. The free ones are too minimal. The paid ones are built for designers — contrast checkers, sliders, cloud sync, team features. If you just want to pick a color and paste a hex into your code, everything is either underpowered or overpriced.

ColorPeek is the first color picker built for people who write CSS for a living.

Free Features

Free forever. More than enough for daily use.

🎯

Menu Bar Color Picker

Click the eyedropper icon in your menu bar, pick any pixel on screen. macOS native NSColorSampler with magnifying glass — 0.5 seconds from click to copy.

📐

4 Format Outputs

One-click switch between HEX (#3B4CFC), RGB (59, 76, 252), HSL (236°, 90%, 61%), and OKLCH — the modern CSS color space.

📋

50-Color History

Every picked color is saved. Search by hex substring. Click any entry to re-copy in your current format. Right-click to delete.

📎

Click to Copy

Pick a color, click it in history — it's on your clipboard. System notification confirms what was copied. No extra steps.

🌓

Light & Dark Mode

Menu bar icon and popup UI auto-adapt to your system appearance. Looks native on every macOS version from Sonoma onwards.

🚀

Launch at Login

Toggle it on, and ColorPeek starts with your Mac. Always there when you need it, invisible when you don't.

🔒

100% Local

Everything stays on your Mac. No data is ever sent anywhere. Not even anonymous usage stats.

🍎

macOS 14+ Native

Built with SwiftUI and AppKit. No Electron, no Catalyst, no web wrapper. Tiny footprint, instant launch.

Pro Features

$19 lifetime. Unlock the developer toolkit.

🎨

8 Format Outputs

All free formats plus Tailwind CSS (closest palette match: indigo-600), SCSS variable ($indigo-600), CSS custom property (--color-indigo-600), and SwiftUI Color (Color(red: 0.23, green: 0.30, blue: 0.99)). No other color picker does this.

🎯

Tailwind CSS Palette — 242 Colors

Full Tailwind CSS 3.4 default palette baked in. Pick any pixel and ColorPeek tells you the closest Tailwind shade. Perfect for Tailwind projects — no more guessing between blue-500 and blue-600.

∞

Unlimited History + Full-Text Search

No 50-color cap. Search by hex fragment, color name, or format. Find that one shade you picked three weeks ago in seconds.

📂

Custom Palettes

Create named palettes to organize colors by project. "Acme Redesign", "Side Project Dark Theme" — keep your colors where they belong. Add colors from history with right-click.

⌨️

Global Keyboard Shortcut

⌘⇧C activates the color picker from anywhere — no need to click the menu bar. The fastest path from idea to hex code.

🖼️

Clipboard Image Color Picking

Screenshot a design mockup (⌘⇧4), then use ColorPeek to sample colors directly from the image. No need to save a file first.

📌

Quick Access Last 6 Colors

Right-click the menu bar icon to see your 6 most recent colors — pick and copy without opening the full panel.

🔑

License Key

One key works on up to 3 Macs. 7-day offline cache — no phoning home on every launch. License server at api.wayknow.tech.

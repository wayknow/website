Built for Trust

Open source, local-first, and transparent by design

CrumbKit was built to fill the gap for a modern, Manifest V3 cookie editor that respects your privacy. Every line of code is open source (MIT). All processing happens locally on your device — zero network requests, zero data collection. No accounts, no ads, no subscriptions. Just a reliable tool that works.

No catch. CrumbKit is genuinely free — no Pro tier, no ads, no data collection. It's part of the WayKnow product family alongside ClearJSON and SnapMark. CrumbKit is our free acquisition tool — we earn revenue from our other products, not by charging for cookie management.

CrumbKit automatically classifies every cookie on the current site using name patterns and a bundled Disconnect.me tracking domain list. It calculates a 0–100 score: higher means fewer trackers. The score is color-coded — green (good), yellow (caution), red (many trackers). All classification happens locally — zero network requests.

Export: JSON (.json), Netscape HTTP Cookie File (.txt — compatible with curl and wget), and cURL command (.sh). Import: JSON and Netscape formats. The parser handles real-world cookie files reliably — including edge cases like special characters, missing fields, and large datasets.

Privacy Policy

Last updated: July 3, 2026

ClearJSON ("the extension", "we", "us", or "our") is an open-source, local-first, zero-tracking JSON viewer. This Privacy Policy explains how we protect your privacy.

TL;DR: We collect absolutely nothing. The code is open source — you can verify this yourself.

1. Information We Collect

1.1 Zero Data Collection

ClearJSON does not collect, store, or transmit any user data to any server. Specifically:

  • No personal information: No account required. We do not collect your name, email address, IP address, or any personally identifiable information.
  • No JSON content: All JSON data you view is parsed and rendered entirely within your local browser. It never leaves your device.
  • No browsing behavior tracking: We do not record which pages you visit or which JSON files you open.
  • No analytics of any kind: No Google Analytics, no usage statistics, no feature usage tracking. Not even anonymous, aggregated data.

1.2 Local Storage

The extension uses Chrome's chrome.storage.local API to store the following data on your device only:

  • User preferences: Theme selection, font size settings, etc.
  • License information: If you purchase the Pro version, license verification data is stored locally.

This data is stored exclusively on your device and is not shared with us or any third party.

2. Open Source & Auditable

ClearJSON's source code is publicly available on GitHub under the MIT license. This means you can:

  • Fully inspect the code and verify that we collect no data
  • Build and install the extension yourself, independent of the Chrome Web Store
  • Fork the code and create your own version

We believe open source is the strongest privacy commitment you can make. You don't have to trust our words — you can read the code.

3. Data Sharing

We do not share any data with any third party — because we collect no data. This answer will not change.

4. Data Security

Since all data resides on your local device, data security is ensured by your device and Chrome browser. We recommend:

  • Keeping Chrome updated to the latest version
  • Using OS-level security features

5. Third-Party Services

ClearJSON does not integrate any third-party services, including but not limited to:

  • Analytics tools (Google Analytics, etc.)
  • Advertising platforms
  • Crash reporting tools
  • CDNs or external resource loading

All features work fully offline. If you purchase the Pro version through Creem, the payment process is handled by Creem — please refer to Creem's privacy policy.

6. Pro License Verification

If you purchase a ClearJSON Pro license, a one-time verification request is made to our license server (api.wayknow.tech) to validate your license key. This is the only network request ClearJSON ever makes.

The verification request contains:

  • Your license key (e.g., CLJ-XXXX-XXXX-XXXX)
  • A random device identifier (generated on your device, not tied to your identity)
  • Your OS and Chrome version (for device management — max 3 devices per license)

This request happens at most once every 7 days. No browsing data, JSON content, or personal information is included. Free users never make any network requests.

7. Permission Justification

ClearJSON requests the following Chrome permissions:

  • storage: Store your preferences locally.
  • Web content access: Detect whether a page you open contains JSON, and if so, render a formatted view. This process is entirely local.

8. How We Differ from Other JSON Tools

Many JSON formatting tools:

  • Upload your JSON data to their servers for formatting
  • Track your usage behavior
  • Inject advertisements

ClearJSON does none of these things. That's exactly why we exist — after the original JSON Formatter was sold and turned into adware, developers needed a tool they could genuinely trust.

9. Children's Privacy

ClearJSON does not collect data from anyone, and therefore does not collect data from children under 13.

10. Changes to This Privacy Policy

If we update this Privacy Policy, we will update this page and revise the "Last updated" date.

11. Contact Us

Privacy Policy

Last updated: July 6, 2026

CookieClear ("the extension", "we", "us", or "our") is a free, open-source, local-first, zero-tracking cookie editor for Chrome. This Privacy Policy explains how we protect your privacy.

TL;DR: We collect absolutely nothing. The code is open source — you can verify this yourself.

1. Information We Collect

1.1 Zero Data Collection

CookieClear does not collect, store, or transmit any user data to any server. Specifically:

  • No personal information: No account required. We do not collect your name, email address, IP address, or any personally identifiable information.
  • No cookie data: All cookies you view, edit, import, or export remain entirely within your local browser. They never leave your device.
  • No browsing behavior tracking: We do not record which sites you visit or which cookies you manage.
  • No analytics of any kind: No Google Analytics, no usage statistics, no feature usage tracking. Not even anonymous, aggregated data.

1.2 Local Storage

The extension uses Chrome's chrome.storage.local API to store the following data on your device only:

  • User preferences: Theme selection (dark/light mode), domain whitelist entries.

This data is stored exclusively on your device and is not shared with us or any third party.

2. Open Source & Auditable

CookieClear's source code is publicly available on GitHub under the MIT license. This means you can:

  • Fully inspect the code and verify that we collect no data
  • Build and install the extension yourself, independent of the Chrome Web Store
  • Fork the code and create your own version

We believe open source is the strongest privacy commitment you can make. You don't have to trust our words — you can read the code.

3. Data Sharing

We do not share any data with any third party — because we collect no data. This answer will not change.

4. Data Security

Since all data resides on your local device, data security is ensured by your device and Chrome browser. We recommend:

  • Keeping Chrome updated to the latest version
  • Using OS-level security features

5. Third-Party Services

CookieClear does not integrate any third-party services, including but not limited to:

  • Analytics tools (Google Analytics, etc.)
  • Advertising platforms
  • Crash reporting tools
  • CDNs or external resource loading

All features, including cookie classification and privacy scoring, work fully offline. The tracking domain list is bundled with the extension — no network requests are ever made.

6. Permission Justification

CookieClear requests the following Chrome permissions:

  • cookies: To read, create, edit, and delete browser cookies at your direction. CookieClear never reads cookies from sites you haven't explicitly opened the popup on.
  • storage: To store your preferences and domain whitelist locally.
  • activeTab: To know which site's cookies to display when you click the extension icon.
  • tabs: To get the URL of the current tab so we know which site's cookies to show. No browsing history is accessed or stored.
  • host permissions: Required by chrome.cookies API to access cookies for the current site. CookieClear only accesses cookies when you open the popup.

7. How We Differ

Unlike other cookie tools:

  • We don't upload your cookies to any server
  • We don't track your browsing behavior
  • We don't inject advertisements
  • We don't charge a subscription for basic features

We built CookieClear to be the tool we wanted to use — one that's free, open source, and genuinely trustworthy. No data collection, no ads, no subscriptions.

8. Children's Privacy

CookieClear does not collect data from anyone, and therefore does not collect data from children under 13.

9. Changes to This Privacy Policy

If we update this Privacy Policy, we will update this page and revise the "Last updated" date.
